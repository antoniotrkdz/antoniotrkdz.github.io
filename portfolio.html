Welcome to Antoniotrkdz GitHub Page.

I am a neuroscientist, and a student programmer.

Here is where you can find my projects.

My first piece of of code is an ImageJ/FIJI macro to stitch together various stacks of images acquired with the Prairie Ultima IV Two-Photon microscopes. For an explanation of what Two-Photon microscopy is please read here

The macro per se does nothing fancy, it just reorganises a set of images in a way that they can then be stitched with the help of TrackEM2 ImageJ/FIJI plugin
Please read the instructions on how to use TrackEM2 for this purpose.
